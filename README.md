# Horney Website ⚡️

## Requirements
- Node v22.18.0
- Bun v1.1.29

### Development
Install with bun:
```sh
bun i
```

Serve with 11ty:
```sh
bun run serve
```

### Build
Build with 11ty:
```sh
bun run build
```

### Deploy
Deploy to GitHub Pages:
```sh
bun run reploy
```
